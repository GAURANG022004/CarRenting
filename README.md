# 🚗 Car Renting System

A simple Java-based Car Rental Management System that allows users to manage cars, customers, rentals, and payments using a layered architecture (DAO, Service, Model).

---

## 📌 Features

* Add and manage cars
* Register and manage customers
* Rent and return cars
* Handle payments
* Input validation and exception handling

---

## 🏗️ Project Structure

```
CarRenting/
│
├── src/
│   ├── carrentalDao/        # Data Access Layer
│   ├── carrentalmodel/      # Model Classes
│   ├── carrentalservice/    # Business Logic
│   ├── exception/           # Custom Exceptions
│   ├── util/                # Utility Classes
│   └── main/                # Main Application Entry
│
├── bin/                     # Compiled Classes
└── .settings/               # IDE Config (Eclipse)
```
